Hylo Pre-March 2026 Deployment Report

Narrow question: did a mainnet Hylo Stability Pool deployment exist before March 27, 2026?

Answer: yes. The current Hylo program IDs were first deployed on mainnet on April 11, 2025. The current binaries were later upgraded on January 28, 2026. The visible pool-account history only starts on March 27, 2026, but that is not the real start of the pool: the first visible pool transaction already shows millions of hyUSD and xSOL in the pool before the transaction begins.

What March 27 Actually Means

The earlier direct pool-account probe only showed history back to March 27, 2026. That looked like a possible launch boundary, but the first visible transaction itself disproves that interpretation.

External Corroboration

Hylo’s public December 2025 Accretion audit matches the on-chain result.

  • The audit lists the same current program IDs: HYEXCH... and HysTab....
  • It marks both as Status: Verified and says the verified build matched what was on-chain.
  • The report timeline places that verified on-chain state in late November 2025.

Bottom Line

  • Yes, a pre-March 27, 2026 mainnet Hylo Stability Pool deployment existed.
  • The current Exchange and Stability Pool program IDs were first deployed on April 11, 2025.
  • The current code was upgraded on January 28, 2026.
  • The current pool accounts were already materially funded before the first visible March 27 pool transaction.
  • So the earlier “late March start” reading was too aggressive. The available public pool-account history is incomplete.
Still open: recovering the full pre-March pool transaction stream. That likely needs an archival indexer, a Hylo frontend/indexer feed, or explorer-side history beyond what direct public RPC pagination exposed.

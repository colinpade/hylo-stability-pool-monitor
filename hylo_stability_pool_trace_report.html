Bottom Line

The screenshot is consistent with a `hyUSD -> xSOL` stability-mode style conversion, and the implied xSOL price in the image is internally consistent with current public xSOL market pricing. I still did not recover those four exact rows as direct public matches, but I did recover a real recurring public pattern:

  • 3,000 recent Hylo Exchange transactions, including decoded SwapStableToLeverEventV1 logs.
  • 1,000 recent Hylo Stability Pool transactions, including decoded RebalanceStableToLeverEvent logs.
  • 1,000 recent transactions for the likely keeper signer and 1,000 for the labeled vault owner.
  • 144 public keeper-buy transactions grouped into 34 time clusters from March 26, 2026 through March 29, 2026 in the current directly retrievable RPC slice.
  • Public xSOL market data from DexScreener and GeckoTerminal, including 5-minute OHLC bars.
Best working interpretation: the tweet is pointing at a real recurring Stability Pool xSOL-acquisition mechanism, but the screenshot itself likely comes from a Hylo-specific aggregation or indexer layer rather than a simple 1:1 explorer row.
Important scope note: official docs, repo dates, and GeckoTerminal pool-creation metadata all show that Hylo / xSOL / hyUSD predate this March window by months. So the public account pagination here is useful for recent pattern work, but it is not a full year-to-date ledger reconstruction.

Recovered Public Mechanism

The clearest public pattern is not a decoded RebalanceStableToLeverEvent row. It is a repeated Stability Pool-linked transaction shape where the keeper wallet spends hyUSD and receives xSOL, while the labeled vault owner loses xSOL in the same transaction.

  • Typical logs include Route or SharedAccountsRoute, SwapV2, Swap, UserWithdraw, and Burn.
  • The strongest public example before the tweet window is 3KwESu...F8bJD at 2026-03-29 10:29 AM PDT.
  • That pre-tweet cluster ran from 2026-03-29 10:28:31 AM to 10:29:36 AM PDT across 18 matching transactions.
  • Largest public clusters in the cached window reached about 133.5 hyUSD and 2,363.2 xSOL, still far smaller than the screenshot rows.
This is the strongest public evidence so far for how the Stability Pool ends up long xSOL in live conditions.
